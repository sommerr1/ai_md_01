# Product Vision — Google AI Mode → Obsidian Saver

## Однострочник

**Сохраняй разговоры из Google AI Mode в Obsidian одним кликом — с форматированием, структурой и контролем над файлами.**

## Проблема

Пользователи ведут исследования и рабочие диалоги в **Google Search AI Mode** («Режим ИИ»), но ценные ответы остаются в браузере: их сложно архивировать, искать, связывать с заметками и использовать в личной базе знаний (Obsidian).

## Решение

Браузерное расширение для **Chrome и Edge**, которое:

- определяет активную вкладку с **AI Mode** в Google Search;
- извлекает **весь тред** (с автоскроллом для подгрузки истории);
- конвертирует в **Markdown с YAML frontmatter**, размеченный секциями `## User` / `## AI`;
- сохраняет в **заранее выбранную папку** vault Obsidian через File System Access API.

## Целевая аудитория

| Сегмент | Потребность |
|---------|-------------|
| Knowledge workers | Архив исследований из AI Mode в Obsidian |
| Исследователи / аналитики | Повторное использование ответов, ссылки, структура |
| Power users Obsidian | Единый vault без ручного copy-paste |

## Ценностное предложение

| Для пользователя | Ценность |
|------------------|----------|
| Один клик | Экономия 5–15 мин на ручное копирование длинного треда |
| Obsidian-ready | `.md` с frontmatter, секциями User/AI, списками, code blocks |
| Контроль | Своя папка, своё имя файла, перезапись при повторном сохранении |
| Без шума | Вне AI Mode — тихий disabled state, без ложных срабатываний |

## Видение на 12 месяцев

**v1 (MVP):** надёжное сохранение текстовых тредов AI Mode → flat folder в Obsidian vault.

**v2:** см. [06-prd-v2.md](./06-prd-v2.md) — выбор формата (MD / MD+YAML / Obsidian MD / HTML / JSON / dual), Gemini + Copilot, медиа в `attachments/`, валидация Mermaid для Obsidian.

**v3+:** настраиваемые шаблоны frontmatter, горячие клавиши, Brave/Arc/Opera, append-only mode.

## Принципы продукта

1. **Obsidian-first** — формат файла важнее pixel-perfect копии браузера.
2. **Explicit scope** — только AI Mode, только текст в v1.
3. **Privacy by default** — данные остаются локально, без телеметрии контента чатов.
4. **Predictable saves** — полный тред, предсказуемое имя, перезапись при том же имени.

## Метрики успеха (North Star)

**Количество успешно сохранённых тредов в неделю на активного пользователя** — прокси реальной ценности архивации.

Вспомогательные: % успешных сохранений, время сохранения (p95 < 20 с), % повторных сохранений (retention).

## Не цели (v1)

- Сохранение Gemini, AI Overviews, обычного поиска Google
- Синхронизация с облаком
- Редактирование чатов внутри расширения
- Скачивание изображений и видео из ответов
